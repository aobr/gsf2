User API
========

``gsf`` exposes three public entry points. Each is usable both from Python
(``from gsf import gsf, gsf_loop, tag_components``) and from the command line
(the ``gsf`` and ``gsf-tag`` commands); see the :doc:`intro` for command-line
examples.
